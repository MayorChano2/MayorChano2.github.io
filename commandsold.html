Mod Commands


bank!warn @Person#6969 'reason' to warn a user

bank!kick 'user' 'reason' to kick a user. Only a mod can run this command

bank!ban 'user' 'reason' to ban a user. Only a mod can run this command

bank!mute '@user' ‘reason’ mute or unmute a member

bank!unban 'userid' to unban a user from the server



General Commands


bank!ping shows the speed of the bot . It works by measuring the time between your command message and its 'Ping?' message. If the ping is above 1000 ms consistently, dm the devs

bank!say 'message' sends your message and then deletes the command message

bank!stats shows some stats about the bot

bank!purge 'number' from 1 up to 100 deletes the amount of messages specified

bank!cpu shows the percent of cpu the bot is using up

bank!report 'error' to send a bug you found. Please send the error code and/or a link to a screenshot of what happened

bank!invite to see my invite link



Music Commands


bank!Play play 'url' or 'search term' to play/choose a song to play

bank!Pause pause to pause the current song

bank!Resume resume to continue playing the current song

bank!Queue queue to see the queue of the server

bank!Clear clear to clear the upcoming queue

bank!Skip skip to skip the current song

bank!Volume volume to change the volume from 1-5

bank!Stop stop to stop the song and end the connection

bank!Leave leave to have the bot leave the channel

bank!Join join for the bot to join your channel

bank!Now Playing np to see what is now playing
